# 15 - Checklist avant partage a un tiers

## A. Hygiene / securite
- [ ] Aucun secret reel dans les docs (`.env`, tokens, cles, mots de passe)
- [ ] Aucune donnee reelle (patients, sites, contacts, identifiants internes)
- [ ] Captures/exports anonymises
- [ ] Logs nettoyes des infos sensibles

## B. Reproductibilite
- [ ] Commandes de demarrage testees de bout en bout
- [ ] Migrations + seeds fictifs verifies
- [ ] Versions outils documentees
- [ ] Variables d environnement documentees
- [ ] Erreurs courantes et correctifs documentes

## C. Fidelite fonctionnelle
- [ ] Regles metier completees
- [ ] Matrice RBAC completee
- [ ] Scenarios E2E documentes
- [ ] Reference UI/UX ajoutee
- [ ] Criteres d acceptation precises

## D. Partage
- [ ] Code source + ce dossier fournis
- [ ] Point de contact defini
- [ ] Session de passation planifiee
- [ ] Canal de support defini (mail, chat, issue tracker)
