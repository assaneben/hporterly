# Support

## Open-source support (GPL-3.0-or-later)

Community support is provided on a best-effort basis through GitHub issues.

- Bug reports: use the bug report issue template
- Feature requests: use the feature request issue template
- Questions: open a sanitized issue/discussion (no real data)

## Commercial support / licensing

Commercial (proprietary) licensing is available for OEM/white-label use,
proprietary redistribution, and closed-source integrations.

- Email: `Couverture@ik.me`
- Alternative: open the `Commercial License Request` issue template

Commercial agreements may include support SLAs, integration assistance,
deployment guidance, and security review support.

## Public channel safety reminder

Never post real patient/facility data, secrets, or internal infrastructure
details in public channels.
